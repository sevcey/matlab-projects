# MATLAB & Simulink Vehicle Systems

This folder includes MATLAB and Simulink projects focused on basic vehicle system modeling, control logic, and simulation.

## Projects

### Vehicle Following Control
A Simulink model that evaluates vehicle following distance and changes the speed command according to safe distance logic.

Keywords: Simulink, Vehicle Systems, Control Logic, Safe Distance, System Modeling

### Lane Keeping Controller
A closed-loop Simulink model that uses PID-based correction for lane keeping behavior.

Keywords: Simulink, PID Control, Feedback Control, Lane Keeping, Vehicle Dynamics

### Safe Distance Control Script
A MATLAB script that simulates basic safe following distance control and visualizes distance and speed behavior.

Keywords: MATLAB, Simulation, Signal Analysis, Control Logic
